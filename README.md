# Elderly Fall and Health Monitoring System

An IoT-based wearable system designed to detect falls and abnormal heart rate conditions in elderly individuals using ESP32.

## Features
- Fall detection using accelerometer
- Heart rate monitoring
- Abnormal heart rate alert
- Emergency buzzer alert
- Remote alert via Telegram (no mobile app required)

## Hardware Used
- ESP32
- MAX30102 Heart Rate Sensor
- MPU6050 Accelerometer
- Active Buzzer
